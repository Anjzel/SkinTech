How to use SkinTech?

How to Properly Upload and Capture Your Face


1. Upload an Image

Click on the "Upload Image" button to select an image from your device.

This allows the user to select an image to upload from their device.

2. Capturing Your Face

Click on the "Capture Face" button to take a picture using your device's camera.

It allows the user to capture their face using their camera.

This is where the system detects the face of the user and submits their captured images. Also, the user must capture three sides (front,left and right) of the face.

This section allows the user to select what type of skin care category they prefer for the recommendation.

3. Results

After submitting the images, it will show you the results page of the personalized recommendation skin care products. This section shows the skin analysis, recommended skin care products, and beneficial ingredients for the type of skin. Lastly, users can download the results on a PDF file.

4. Reports

This is the file of the downloaded facial skin care report, the content includes skin type & sensitivity, beneficial ingredients, skin analysis and recommended products.
